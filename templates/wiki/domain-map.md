## How to add a domain

1. Create `wiki/domains/<slug>/overview.md` (architect lens, pillars, progress, glossary).
2. Optional: `wiki/domains/<slug>/guide.md` (concept index + mental model).
3. Per domain keep **overview + guide only** (no `index.md` / legacy `framework` / `graph`).
4. Add a row above; add a row to vault [[overview]].

**Table rule:** In markdown tables, do **not** use `[[path|alias]]` — the `|` breaks columns. Use `[[domains/<slug>/overview]]` only.
